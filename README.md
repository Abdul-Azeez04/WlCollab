# WlCollab

Wallet Collector Website for Collab Managers.

X: @WlCollab

## Features
- Multi-chain wallet collection (EVM, Solana, Aptos, Sui, BTC)
- Community codes per collab manager
- GTD / FCFS quota enforcement
- Tiered allowlists (Allowlist 1, Allowlist 2, etc.)
- Duplicate wallet blocking (optional per project)
- CSV export
- Creator admin dashboard
- Netlify deploy-ready
